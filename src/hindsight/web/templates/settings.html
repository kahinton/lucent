API Access

Use these endpoints to integrate with Hindsight:

MCP Server http://localhost:8765/mcp
REST API http://localhost:8766/api
API Docs http://localhost:8766/api/docs →

Example Usage

# Search memories
curl "http://localhost:8766/api/search?query=python"

# Get a specific memory
curl "http://localhost:8766/api/memories/{memory_id}"

# Create a memory
curl -X POST "http://localhost:8766/api/memories" \
  -H "Content-Type: application/json" \
  -d '{"username": "user", "type": "technical", "content": "..."}'
