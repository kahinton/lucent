Hooks are safe agent middleware

They add approved context around model and tool lifecycle events. Command hooks can run approved shell commands/scripts out-of-process with timeout and output limits. The built-in memory hook is on by default for every agent, so agents can remember relevant repo knowledge when they touch files.
