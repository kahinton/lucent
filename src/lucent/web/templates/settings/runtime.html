How these settings are applied

Runtime values saved here live in the database, so they are included with normal backups and shared by the workspace. If a setting is reset or has never been saved, Lucent falls back to its environment variable, then to the built-in default.

Bootstrap and credential settings are shown for visibility, but locked or hidden when they must remain in environment, Secrets, or Connections.
