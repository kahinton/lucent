API Access

Use these endpoints to integrate with mnemeMCP:

MCP Server http://localhost:8766/mcp
REST API http://localhost:8766/api
API Docs http://localhost:8766/api/docs →

Example Usage with API Key

# Search memories (with API key)
curl "http://localhost:8766/api/search?query=python" \
  -H "Authorization: Bearer mcp_your_api_key_here"

# Create a memory
curl -X POST "http://localhost:8766/api/memories" \
  -H "Authorization: Bearer mcp_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{"type": "technical", "content": "How to fix X..."}'
